fundamentos

tecnologias utilizadas

Este projeto demonstra o poder das três tecnologias nativas da web, combinadas com boas práticas e atenção ao detalhe. (esse texto e os abaixo foram feitos com ia pra tapar buraco)

HTML5

A espinha dorsal de toda página web. Define a estrutura semântica do conteúdo, garantindo acessibilidade, SEO e organização lógica dos elementos.

Estrutura

CSS3

Responsável por toda a aparência visual. Variáveis, Flexbox, Grid, animações, glassmorphism e media queries constroem a experiência estética e responsiva.

Estilo

JavaScript

O motor de interatividade. Gerencia scroll suave, animações de entrada via Intersection Observer, estado da navbar e qualquer comportamento dinâmico da página.

sobre este projeto

fiz o que consegui revendo o material passado em aula e videos no youtube

01

objetivo do portfolio

apresentar habilidades tecnicas de forma clara e elegante, demonstrando dominio de HTML, CSS e JavaScript (forma elaborada de dizer "prova")

02

organização do codigo

tres arquivos separados — HTML, CSS e JS — Código limpo e fácil de navegar

03

responsividade

adaptação fluida para desktop, tablet e celular usando flexbox, grid e media queries. nenhum elemento quebra em nenhuma resolucao (acho)

04

facil personalizacao

variaveis CSS centralizadas no :root permitem trocar toda a paleta de cores com apenas alguns valores (comentarios mostram exatamente o que editar)

05

preparado para expansao

A estrutura modular facilita adicionar novas secoes — projetos, habilidades, contato — sem quebrar o layout existente, basta seguir os padroes ja estabelecidos no codigo
